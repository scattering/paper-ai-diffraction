# TACC Archive Scripts

These scripts are preserved as historical campaign launchers from the original April 2026 paper run sequence.

They are intentionally not the preferred public interface for this repository because they contain:

- hard-coded TACC accounts, partitions, and scratch paths
- environment activation assumptions specific to the original campaign
- operational details that are useful for provenance but not for a portable paper repo

Use the canonical top-level scripts in `../` (repo scripts/) for the paper-facing interface. Use this archive only when you need an exact reference for how the original TACC jobs were launched.
